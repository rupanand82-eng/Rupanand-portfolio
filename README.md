## 🛠️ Technology Stack

## 💎 Local Development Setup

To boot this premium full-stack portfolio locally:

1. **Clone & Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configure Environment Secrets**:
   Create a `.env` in the root (matching `.env.example` template guidance):
   ```env
   GEMINI_API_KEY="your_google_ai_studio_api_key"
   APP_URL="http://localhost:3000"
   ```

3. **Launch Live Playgrounds**:
   ```bash
   npm run dev
   ```

4. **Verify Lints & Compile Assets**:
   ```bash
   npm run lint
   npm run build
   ```
